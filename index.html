Criação Do HTML

(voltar)

Em 1990, Tim Berners-Lee, um cientista da computação britânico, concebeu o HTML durante seu trabalho no CERN (Organização Europeia para Pesquisa Nuclear). Sua intenção inicial era facilitar o compartilhamento de informações e documentos entre pesquisadores e cientistas, resultando na criação dessa linguagem simples e acessível. Somente apos essa criação o mundo começou a ver uma nova forma de comunicação, que ajudou muito a comunicação global de forma simplificada e rapida, sem a necessidade de aguardar dias e mais dias para chegada de cartas. Que era o meio de comunicação mais utilizados antes desse grande avanço tecnologico.

O HTML(HyperText Markup Language)é um conjunto de elemtntos. Podendo ser eles, textos, audios, videos, imagens, etc. Quando conectacodos aos servidores. Formão uma ampla rede de informações.

Hoje quando abrimos nossos Browsers (Navegadores), acessamos muitos sites todos os dias. E esses sites, em sua maior parte são estrutuas HTML, combinada com um conjuntos de outros recursos. Que quando combinados e utilizados em conjunto, formão um site, mais elegante e estéticamente bonito, a maior parte dos sites modernos são criados dessa forma.

Em grande partes dos sites que temos hoje, são um estrutura formada pelos seguintes recurso: HTML, Java Script e CSS. O HTML por sua vez é usado para criar o esqueleto de uma pagina. Java Script geralmente é usado para criação de codigos dinamicos, como os botões, integrando com servidoes e bancos de dados. CSS é utilizado para estilizar e formatar graficamente qualquer pagina HTML.

Como se Comporta a Estrutura

(voltar)

HTML usa formas chamadas (tags), Todo documento HTML possui marcadores (tags), que são palavras delimitadas por parênteses angulares (< e >); esses marcadores são os comandos de formatação da linguagem. Um elemento é composto por um nome de tag, atributos, valores e pode conter filhos (que podem ser outros elementos ou texto). Os atributos modificam os resultados padrão dos elementos, enquanto os valores caracterizam essas modificações. Aqui está um exemplo de um elemento simples que não possui filhos

Com um documento HTML podemos criar varios tipos de sites. De varias formas diferentes, desde simples Landing pages, Blogs até grandes sites de e-commerce.

O codigo de um documento HTML pode ser escrito de qualquer editor de texto comuns, como um simples, Bloco de Notas do Windows, ou o TextEdit do Macintosh. Para facilitar a produção de documentos, no mercado existem editores HTML específicos, com recursos sofisticados, que facilitam a realização de tarefas repetitivas, inserção de objetos, elaboração de tabelas e outros recursos.

Adaptabilidade e Evolução Constante

(voltar)

Desde sua criação, o HTML tem evoluído continuamente para acompanhar as demandas e inovações tecnológicas. Novas versões e especificações têm sido desenvolvidas para suportar funcionalidades avançadas, como vídeos incorporados, interfaces interativas e responsividade móvel, mantendo-se como uma linguagem robusta e essencial para o desenvolvimento web moderno.

Em resumo, o HTML não é apenas um conjunto de tags e atributos; é a linguagem que possibilita a comunicação e interação digital em escala global. Com sua estrutura clara e flexível, o HTML continua a ser a base sobre a qual a internet é construída, capacitando desenvolvedores a criar experiências únicas e acessíveis para todos os usuários da web.
